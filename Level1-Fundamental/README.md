# Level 1 - Fundamental (Wajib Kuasai Dulu)

Level ini mencakup dasar-dasar yang wajib dikuasai sebelum masuk ke pentesting lanjutan. Fokus pada Linux, networking, dan scripting dasar.

**Pemahaman:**

- Process management: ps, kill, top

2. Ikuti tutorial basic Linux command
3. Praktik daily di terminal
